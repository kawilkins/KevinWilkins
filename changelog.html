What is a "Change Log"?

A change log is a list of changes made to a project such as a website, application, or computer program. Typically this list will include records of new features, bug fixes, etc. Anything that has changed can be considered part of the change log. Changes are usually logged in chronological order and likely include a heading that indicates the version.
